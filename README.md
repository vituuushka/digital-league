# Приложение для рисования

## Задание

Необходимо на React создать холст и две кнопки  

— Первая кнопка должна активировать кисть для рисования на этом холсте  

— Вторая кнопка должна активировать "Ластик"  

— Приложение должно быть разбито на компоненты  

— Разрешается использование стейт менеджеров

## Технологии

- **React 18** + **TypeScript** (основной стек)
- **Vite** (сборщик проекта)
- **Zustand** (управление состоянием)
- **React Icons** (иконки в интерфейсе)
- **Canvas API** (рисование на холсте)
